Replaces twitter usernames and hashtags automatically in posts and excerpts;

== Description ==

This plugin will look for twitter @usernames and #hashtags and replace them automatically with the corresponding links; E-mail addresses will be ignored, as expected;

Any suggestion, please, send to tweetfy@filipekiss.com.br
